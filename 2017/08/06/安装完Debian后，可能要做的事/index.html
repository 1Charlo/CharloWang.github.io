安装完Debian后，可能要做的事

发表于 2017-08-06 | 更新于 2018-11-19 | 阅读次数：

1、为终端启动添加快捷键

在“设置”->“快捷键”->“自定义”中添加，名字填 Terminal，命令填 /usr/bin/gnome-terminal，完成后保存并返回，然后添加快捷键，一般设置为 Ctrl+Alt+T。

4、添加数据源

国内比较好用的源包括163、中科大、清华等，

在终端中输入： sudo gedit /etc/apt/sources.list，将上面的任意一个源添加到文件中，保存并退出，然后在终端中输入： sudo apt-get update，至此数据源就已经配置成功了。

5、安装基本编译环境

在终端中输入： sudo apt-get install gcc build-essential。

6、安装中文输入法（google输入法）

安装完成之后，到fcitx软件中选择输入法，并在全局配置–>高级中激活输入法。在输入法软件中配置fcitx。

7、创建pppoe上网连接

8、自动挂载NTFS分区

首先查看分区情况：sudofdisk -l,
然后为想要挂载的分区创建挂载点，在 /media/用户名/ 下创建文件夹，如hd1、hd2、hd3…
然后 sudo gedit /etc/fstab，在文件尾部添加：
/dev/分区号 /media/用户名/hd1 ntfs-3g defaults,locale=zh_CN.UTF-8 0 0

9、将本地ISO镜像作为软件安装源

如果你当前处于无网的状态，可以使用本地的镜像作为软件安装源。

10、主题安装

去gnome主题网站下载主题后，解压后放到/usr/share/themes中。安装gnome-tweak-tool工具，在优化工具–>扩展，中打开user主题。然后在外观–>主题中切换即可

11、安装Dock（底部导航栏，类似于MacOS的Dock）

Linux下好看的Dock有好几个，比如：Docky、Avant Window Navigator、Cairo-Dock等，这里我用Dash to Dock。
打开“优化工具”，找到“扩展”->”Shell 扩展”，搜索并下载安装Dock，安装完成后即可看到界面的变化。
