HEXO+Github搭建博客

发表于 2017-06-12 | 更新于 2018-11-25

建立Repository

建立与自己的github用户名相同的仓库，仓库名的固定写法为【your_user_name.github.io】
仓库创建好后以备后用

安装git

git安装完成后，生成SSH密钥：

ssh-keygen -t rsa -C “你的邮箱地址”，然后一直回车。
去用户目录下的.ssh文件夹中，找到两个文件id_rsa和id_rsa.pub。
打开id_rsa.pub，复制全部内容，打开https://github.com/settings/ssh ，Add SSH key，将刚复制的内容粘贴进去。

安装Node.js

安装完Node后，记得配置Node的国内镜像【原因你懂的。。。】
推荐使用淘宝镜像：npm config set registry https://registry.npm.taobao.org

安装Hexo

上面的两个步骤【git、node.js】都是为了安装和使用hexo做准备。
在一个自己喜欢的目录下创建一个文件夹【名字随意】，作为hexo的安装、配置目录。
然后执行命令：npm install -g hexo 【安装hexo】
在刚刚的目录下，打开git bash heare，执行init命令初始化hexo：hexo init ，这个目录就是网站的根目录。
